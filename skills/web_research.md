## Overview

Use this skill when a goal requires gathering external information before taking action. Produces a structured synthesis, not a raw dump.

## Steps

1. **Identify the core question** — restate what you need to know in one sentence before searching.
2. **Search with 2-3 distinct queries** — vary angles (definition, examples, recent developments, criticism).
3. **Skim result summaries** — read titles and snippets before fetching full pages; skip paywalled or dated sources.
4. **Fetch 2-4 high-quality pages** — prioritize primary sources, academic papers, official docs over aggregators.
5. **Extract key claims** — bullet the 5-10 most relevant findings with source attribution.
6. **Identify conflicts** — note where sources disagree and flag the most credible position.
7. **Synthesize** — write a 2-3 paragraph summary that answers the original question directly.
8. **Flag gaps** — list what you still don't know and whether it blocks the goal.

## Quality gates

- At least 2 independent sources for any factual claim you rely on.
- Do not copy-paste; paraphrase and attribute.
- If the goal is time-sensitive, check publication dates — prefer sources from the last 12 months.
