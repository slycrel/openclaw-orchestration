## Recommended Core Stack (Highest Evidence Tier)

For a pragmatic daily stack prioritizing RCT support and safety:

| Tier | Compounds | Rationale |
|---|---|---|
| **Foundation** | Creatine 3–5 g + Caffeine 100–200 mg + L-Theanine 200 mg | Broadest RCT base; safest profile; immediate + sustained effect |
| **Build** | Bacopa 300 mg + Rhodiola 200–400 mg (cycled) | Delayed recall + fatigue resistance; 4–8 wk onset for Bacopa |
| **Neuroprotection** | Omega-3 1–3 g + Phosphatidylserine 300 mg | Long-term brain health; FDA-qualified synergy pair |
| **Optional Add** | Ashwagandha 300–600 mg (if stress is a cognitive bottleneck) | Cortisol → cognition pathway; strong RCT for stress |
| **Lower Priority** | Citicoline 250 mg (if not using Alpha-GPC) | Moderate evidence; safe; do not combine with Alpha-GPC |
| **Low Priority** | Alpha-GPC, Lion's Mane | Weak RCT in healthy adults; use only if specific rationale exists |

## MAOI / High-Risk Interaction Summary

**Full stack contraindicated with MAOIs** (Rhodiola, Caffeine both pose hypertensive/serotonin risk).  
**Thyroid patients:** Ashwagandha requires physician oversight.  
**Anticoagulant patients:** Omega-3 >3 g/day requires INR monitoring.  
**Kidney disease:** Creatine contraindicated.  
**Pregnancy:** Ashwagandha, Bacopa, Rhodiola — all avoid.
